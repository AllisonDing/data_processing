# data_processing

End-to-end **structured + unstructured** data processing on real Amazon datasets,
accelerated with **RAPIDS (cuDF / cuML)** and surfaced through a **Streamlit** dashboard.

Two notebooks drive the pipelines, and `run.sh` executes both back-to-back before
launching the dashboard:

| # | Notebook | Pipeline | Output |
|---|---|---|---|
| 01 | [`01_structured_amazon_products.ipynb`](01_structured_amazon_products.ipynb) | Polars + cuDF GPU aggregations over 139 per-category CSVs, currency/rating cleanup, Wilson-score rating | `data/amazon/amazon_products_clean.parquet` + overview plots |
| 02 | [`02_unstructured_amazon_reviews.ipynb`](02_unstructured_amazon_reviews.ipynb) | Nemotron-1B embeddings → Milvus vector store → BERTopic clustering + UMAP | `data/amazon_reviews/amazon_reviews_sentiment.parquet` + BERTopic HTML panels |

The Streamlit app ([`app.py`](app.py)) reads those artifacts and renders them
across seven tabs: Overview, Products, Reviews EDA, Sentiment Timeline,
Embedding Space, Topics (BERTopic), and Document Datamap.

## Datasets

- **Notebook 01** — [Kaggle: `lokeshparab/amazon-products-dataset`](https://www.kaggle.com/datasets/lokeshparab/amazon-products-dataset). Pulled via the Kaggle CLI in the first cell (`~/.kaggle/kaggle.json` must be configured).
- **Notebook 02** — [Amazon Reviews 2023 (McAuley Lab, UCSD)](https://amazon-reviews-2023.github.io/). Downloaded with `wget` from the first cells; uses the `Appliances`, `Electronics`, and `Clothing_Shoes_and_Jewelry` review categories.

Both data trees are git-ignored — they are regenerated by running the notebooks.

## Prerequisites

- NVIDIA GPU with CUDA 13 drivers (RAPIDS 25.12 targets `cuda-version>=13.0,<=13.1`).
- `conda` (Miniforge / Mambaforge recommended).
- Kaggle API token for notebook 01 — see [Kaggle API docs](https://www.kaggle.com/docs/api).

## Execution

**Step 1** — Install the conda environment:

```bash
conda create -n rapids-25.12-python-3.13 -c rapidsai -c conda-forge \
    cudf=25.12 cuml=25.12 python=3.13 'cuda-version>=13.0,<=13.1' \
    dash jupyterlab
```

**Step 2** — Activate it:

```bash
conda activate rapids-25.12-python-3.13
```

**Step 3** — Run the full pipeline + dashboard:

```bash
./run.sh
```

`run.sh` executes both notebooks via `nbconvert` (outputs are written to a temp
directory, so the source `.ipynb` files stay unmodified), then launches
`streamlit run app.py`.

To skip the notebook execution and just open the dashboard against artifacts
already on disk:

```bash
streamlit run app.py
```
